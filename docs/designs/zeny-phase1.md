## Vision

### 10x Check
The 10x version of Phase 1 isn't more code — it's a demo so compelling that the user records their screen and posts it on X. The live cycle dashboard showing personas changing their minds in real-time, ending with a shareable summary that proves the agent learned. That's what turns a CLI tool into a viral moment.

## Accepted Scope (added to this plan)
- Context reset boundary: truncate lessons to last 3 entries + persona memory to last 5 cycles per cycle
- Live cycle dashboard: formatted terminal output showing generation, simulation, scoring, persona quotes, and decisions in real-time
- Session summary: auto-generated summary.md with best version, learning curve, key lessons, persona evolution
- Directory-driven personas: glob personas/*.md, ship 5 defaults, users add/remove freely

## Deferred to TODOS.md

## Additional Scope (from Outside Voice)
- Baseline control run (`--control` flag): run same seed without lessons/memory to falsify placebo learning
- Metric gaming guardrails: separate scoring rubric from generator prompt — generator sees reactions but NOT sub-metric weights
- Falsifiable success metric: v5 composite score must be ≥20% higher than v1 across 3 independent runs with different seeds (replaces "dev friend says whoa")
